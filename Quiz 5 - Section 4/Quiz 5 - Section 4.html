(d) Flag subset of data and plot

  • Create a boolean colum (True/False) "bool_cond" which checks whether
    "growth_cb" had a value greater than 15 or less than -15
  • Generate a line plot with the "growth_cb" series, label the axes and the title
  • Use the command plt.fill_between() as in Lecture 22 to create a shaded region
    for time periods where "bool_cond" is true.

bitcoin [ "bool_cond" ]     = bitcoin [ "growth_cb" ] > 15 
bitcoin [ "bool_cond1" ]    = bitcoin [ "growth_cb" ] < -15

plt.plot    ( "date", "growth_cb", data = bitcoin )
plt.xlabel  ( "Time" )
plt.ylabel  ( "Growth of 1 CB to USD" )
plt.title   ( "Extreme Growth of 1 CB to USD Over Time" )

vec_y = bitcoin [ "growth_cb" ]
plt.fill_between ( x = bitcoin [ "date" ],
                 y1 = vec_y.min (),
                 y2 = vec_y.max (),
                 where = bitcoin [ "bool_cond" ] | bitcoin [ "bool_cond1" ] ,
                 alpha = 0.8, color = "red" )

plt.show()

(e) Aggregate dataset by time period

  • Create an object "month_config" that uses pd.Grouper(), setting the key to
    the date column and freq to "m" (month)
  • Follow the instructions in Lecture 21 to compute an aggregate dataset
    by month which computes the standard deviation of "cb_to_usd"
  • Display the dataset

# Write your own code

month_config = pd.Grouper( key = 'date', freq = 'm' )

monthlydata = ( bitcoin
               .groupby( month_config ) 
               .agg( cb_std = ( "cb_to_usd" , "std" ) )
               .reset_index ( ) )
